## Problemas comuns
Caso tenha problemas com certificado SSL o comando abaixo atualiza o certificado de segurança
### Windows
- pip install --upgrade certifi

### Linux
- sudo apt-get update
- sudo apt-get install --reinstall ca-certificates
- sudo update-ca-certificates

#### Após iniciado o ambiente virtual

- pip install --upgrade certifi
